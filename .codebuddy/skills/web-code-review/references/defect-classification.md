# 缺陷分类表

来源：Code Analysis Guide for Web Code Review

## 缺陷大类

| 大类 | 英文 | 说明 |
|------|------|------|
| 代码逻辑类 | Code Logic | 函数/方法逻辑问题 |
| 视觉呈现类缺陷 | Visual Presentation | CSS/样式/布局问题 |
| 交互体验类缺陷 | User Interaction | 事件/反馈/表单验证问题 |
| 性能表现类缺陷 | Performance | 内存泄漏/渲染性能/算法效率 |
| 内容准确性类缺陷 | Content Accuracy | 文本/数据/日期/翻译错误 |

## 严重程度

| 级别 | 说明 | 举例 |
|------|------|------|
| Major（严重） | 必须修复 | 安全漏洞、数据丢失风险、核心功能崩溃 |
| General（一般） | 按需报告 | 代码质量、小性能问题、不一致模式 |
| Suggest（建议） | 按需报告 | 最佳实践、代码风格、微优化 |

## 详细分类映射

### 代码逻辑类

| source | type | subtype | 说明 |
|--------|------|---------|------|
| 编码 | 函数、模块接口_编码 | 函数功能不单一 | 函数做了过多的事情 |
| 编码 | 函数、模块接口_编码 | 返回值处理、return | 缺少/错误的返回值处理 |
| 编码 | 函数、模块接口_编码 | 代码冗余、逻辑不合理 | 冗余代码或不合理逻辑 |
| 编码 | 函数、模块接口_编码 | 形参和实参类型匹配、参数说明及校验 | 缺少参数校验 |
| 详细设计 | 条件、分支、循环 | 判断错误 | 条件判断逻辑错误 |
| 详细设计 | 程序结构 | 算法问题 | 算法实现错误 |

### 性能表现类

| source | type | subtype | 说明 |
|--------|------|---------|------|
| 编码 | 资源类 | 内存、fd、句柄、锁管理 | 内存泄漏（如事件监听器未清理） |
| 概要设计 | 软件效率性能 | — | 性能问题 |
| 详细设计 | 程序结构 | 实现过于复杂 | 过度复杂的实现 |

### 交互体验类

| source | type | subtype | 说明 |
|--------|------|---------|------|
| 详细设计 | 输出信息_详细设计 | 提示信息错误 | 错误的用户提示信息 |
| 编码 | 输出信息_编码 | 格式不准确 | 消息格式问题 |

### 内容准确性类

| source | type | subtype | 说明 |
|--------|------|---------|------|
| 编码 | 输出信息_编码 | 日志描述、拼写不准确，遗漏关键信息 | 拼写错误或描述不准确 |
| 需求分析 | 需求描述不清晰 | — | 需求理解偏差导致的代码问题 |
